===========================================
Universal Scene Description Python Snippets
===========================================

This is an index of Python code snippets for Universal Scene Description (USD). We have categorized the the snippets by topics and each snippet is titled by the task that it is used for. Where applicable, we include code snippets using Kit USD wrappers for developing within Omniverse and also code snippets using the original USD API. New USD developers will find this useful for finding the appropriate API used to accomplish a task and using the `USD C++ documentation <https://graphics.pixar.com/usd/dev/api/index.html>`_ to learn more about the API. Experienced developers can use this index for quick code lookups.
